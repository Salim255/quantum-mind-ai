Photons and Polarization

Quantum behavior is not limited to microscopic particles like electrons. A similar effect can be observed using light, through an experiment with polarized filters. Light can be thought of as having different polarization directions. A polarizing filter only allows light aligned with its orientation to pass through, while blocking other directions.

Key experiment (three filters)

  1. If two polarizing filters are aligned, light passes through both easily.
  2. If one filter is rotated by 90°, no light passes through the pair.
  3. Surprisingly, if a third filter is inserted between them at 45°, some light begins to pass through again.

This is unexpected, because adding an extra filter increases the amount of light that gets through.

Quantum interpretation

This behavior mirrors quantum measurements:

  • A polarizing filter acts like a measurement device.
  • Each filter asks a “yes/no” question about the photon’s polarization.
  • If the answer is “yes”, the photon passes through and can be measured again.
  • If the answer is “no”, the photon is absorbed and the process stops.

Connection to quantum measurement

With two aligned filters, we are effectively repeating the same measurement, so the result is consistent. With a 90° rotation, we are asking a different but related question, which produces a different outcome. With the 45° filter inserted, we introduce an intermediate measurement that changes the state of the photons, allowing some to pass through the final filter even though they would not have passed directly.

Key idea

Measurements in quantum systems do not just reveal information — they actively change the state of the system, which affects future measurements.

Why it matters

This experiment shows that quantum behavior can be observed in simple optical setups and is not limited to abstract or microscopic particle experiments. It directly connects to the same principles used in electron spin and qubits.
